# AI_tech_HR_model
Проект представляет собой прототип интеллектуальной системы прогнозирования потребности в массовом подборе персонала.

## Состав проекта
1.	data_hist.xlsx – синтетический набор данных, имитирующий задачу массового подбора персонала. Реальные персональные и корпоративные данные не используются.
2.	01_forecast_u_q.ipynb - ML-модуль прогноза коэффициентов текучести сотрудников (U) и выбытия сотрудников-стажеров (Q)
3.	forecast_u_q_2026.xlsx – набор данных, полученный в ходе запуска ML-прогноза коэффициентов текучести на 2026 год
4.	02_business_staffing_model.ipynb – расчетная модель кадровой потребности
5.	hr_results.xlsx – результат запуска модели потребности в персонале.

## Логика решения
•	На исторических данных 2023–2025 обучается ML-модуль.

•	На backtest 2025 года сравниваются модели Ridge Regression, Random Forest, CatBoost и LightGBM и выбирается подходящая для каждого параметра: для прогноза U - CatBoost-модель, для прогноза Q - LightGBM-модель.

•	Прогнозные коэффициенты передаются в расчетную модель.

•	Расчетная модель формирует прогноз дефицита, потребности в найме, контактах и соискателях.
