# AI_tech_HR_model

Проект представляет собой прототип системы прогнозирования потребности в массовом подборе персонала. Решение объединяет ML-модуль прогноза коэффициентов текучести и расчетную бизнес-модель, которая переводит прогнозные коэффициенты в понятные HR-показатели: дефицит основного штата, потребность в найме, количество контактов и соискателей.

## Состав проекта

1. data_hist.xlsx - синтетический набор данных, имитирующий задачу массового подбора персонала. Реальные персональные и корпоративные данные не используются.
2. 01_forecast_u_q.ipynb - ML-модуль прогноза коэффициентов текучести сотрудников (`U`) и выбытия сотрудников-стажеров (`Q`).
3. forecast_u_q_2026.xlsx - результат запуска ML-модуля: прогнозные коэффициенты `U_pred` и `Q_pred` на 2026 год.
4. 02_business_staffing_model.ipynb - расчетная модель кадровой потребности.
5. hr_results.xlsx - бизнес-ориентированный результат запуска расчетной модели.

## Логика решения

1. На исторических данных за 2023–2025 годы обучается ML-модуль.
2. На backtest 2025 года сравниваются модели Ridge Regression, Random Forest, CatBoost и LightGBM.
3. Для прогноза текучести основного штата (U) выбрана CatBoost-модель.
4. Для прогноза выбытия сотрудников-стажеров (Q) выбрана LightGBM-модель.
5. Прогнозные коэффициенты (U_pred) и (Q_pred) передаются в расчетную модель.
6. Расчетная модель формирует прогноз дефицита, потребности в найме, контактах и соискателях.

## Как запустить проект в Google Colab
Ноутбуки выполнены в Google Colab. В коде используются пути к файлам в папке `/content/`, поэтому перед запуском нужно вручную загрузить необходимые Excel-файлы в среду Colab.
